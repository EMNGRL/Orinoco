            You could make use of an iframe and submit the data using window.parent.postMessage('localStorage', JSON.stringify(localStorage)) and window.addEventListener('message', (event) => { /* ... */ }) to submit it, from one origin to another origin.

            Otherwise the data is protected from cross origin access, as mentioned in the article and it is not possible for origin A to access the data from origin B.
